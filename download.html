Minecraft: Java Edition

  • Original PC version
  • Access to mods and custom servers
  • Frequent snapshots of upcoming features
  • Cross-play with other Java Edition players
  • Best for technical players and modders

System Requirements:

Windows 10/11, macOS, or Linux

Minimum 4GB RAM

Minecraft: Bedrock Edition

  • Unified version across devices
  • Cross-play with consoles and mobile
  • Access to Minecraft Marketplace
  • Better performance on lower-end devices
  • Best for casual players and families

Available Platforms:

Windows 10/11, Xbox, PlayStation, Switch, Mobile

    Frequently Asked Questions

    Can I get both editions?

    Yes! If you buy Java Edition from the official Minecraft site, you get Bedrock Edition for Windows for free, and vice versa.

    Which edition updates first?

    Both editions now receive updates at the same time as part of the "Better Together" initiative.

    Can I transfer my worlds between editions?

    There are third-party tools available, but official world transfer between Java and Bedrock is limited.
